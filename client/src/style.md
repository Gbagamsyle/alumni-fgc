. Brand Colors

Usage:

App background

Neutral Colors

Border / Divider: #E5E7EB

2. Color Usage Rules

Never use pure black (#000000)

Background should always be #F6FAF7

Buttons must use:

Background: #86EE60

Text: #1F2937 or #FFFFFF

Hover states:

Darken primary color slightly

Example hover:

background-color: #6fda4f;

3. Typography
Font Family

Primary: Inter, Poppins, or Roboto

Fallback:

font-family: Inter, system-ui, sans-serif;

Font Sizes

Page Title: 32px

Section Title: 24px

Subheading: 18px

Body Text: 14px–16px
